# Fon rasmini qo'shish

Ushbu faylni shu yerga qo'ying:
`assets/background.jpg`

## Rasm talablari:
- Format: JPG, PNG
- Tavsiya etiladigan o'lcham: 1080x1350 yoki katta
- Romantik, chiroyli dizayn

## Qanday qo'shish:
1. O'zingizning rasmingizni tayyorlang
2. Uni `background.jpg` deb nomlang  
3. `assets/` papkasiga qo'ying
4. Botni qayta ishga tushiring: `npm run build && npm run start:prod`

Agar rasm topilmasa, bot avtomatik ravishda gradient fonni ishlatadi.
